## Technical Achievements

### 1. Gaussian Representation
```python
Gaussian3D:
  - position: (3,) world coordinates
  - covariance: (3, 3) positive definite
  - color: (3,) RGB [0,1]
  - opacity: [0,1] alpha value
```

### 2. Covariance Projection
Implemented the key formula from Kerbl et al.:
```
Σ' = JW Σ W^T J^T

where:
- J: projection Jacobian (2×3)
- W: view transformation (3×3 rotation)
- Σ: 3D covariance (3×3)
- Σ': 2D screen covariance (2×2)
```

### 3. Rendering Pipeline
- Project 3D Gaussians to 2D image space
- Sort by depth (back-to-front for α-blending)
- Rasterize with per-pixel Gaussian evaluation
- α-blending: C = Σ ci αi Gi Ti

### Performance

- Bottleneck: Nested loop rasterization

## Current Limitations

1. **Rendering quality**: Basic point-splatting, not full Gaussian evaluation
2. **Performance**: O(N×H×W) complexity, needs tile-based optimization
3. **No SH**: Missing spherical harmonics for view-dependent color
4. **No training**: No optimization/adaptive density control yet
