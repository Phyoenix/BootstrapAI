## 产品概述

**ComputeBench** - 轻量级计算性能基准测试工具  
**当前版本**: v0.1.0  
**状态**: 可运行，已产出第一份性能报告

- 浮点性能: ~6800 GFLOPS
- 整数性能: ~24000 GIOPS  
- 内存带宽: ~30 GB/s

## 开发过程中的学习

### 技术学习

1. **OpenMP并行编程**
   - 使用`#pragma omp parallel for reduction(+:sum)`实现循环并行
   - 学习reduction子句处理共享变量竞争
   - 理解线程调度对性能测试的影响

2. **C++17现代特性**
   - `std::chrono`高精度计时
   - 结构化绑定（在result.cpp中）
   - 文件系统操作（未来版本使用）

3. **性能测试方法论**
   - 预热(warmup)的重要性：避免冷缓存影响
   - 多次采样取平均：减少随机波动
   - 标准差计算：评估结果稳定性

### 架构设计决策

1. **插件式架构**
   - 使用宏`REGISTER_BENCHMARK`自动注册测试
   - 基类Benchmark定义统一接口
   - 便于未来添加新测试类型

2. **双格式输出**
   - JSON：机器可读，用于历史对比
   - Markdown：人类可读，即时报告

3. **统计指标**
   - 不仅报告平均值，还报告min/max/stddev
   - 让用户了解结果的稳定性

## 如何使用这个产品

```bash
cd ComputeBench
make          # 编译
./bin/cb      # 运行全部测试
./bin/cb --test=fp_ops  # 运行指定测试
```

查看报告：
- JSON数据: `results/latest.json`
- Markdown报告: `reports/latest.md`
